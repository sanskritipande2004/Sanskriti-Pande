Sorting Visualizer

Description:
The Sorting Visualizer project allows users to visualize various sorting algorithms in action. By representing the sorting process with dynamic bars or elements, it helps users understand how algorithms like Bubble Sort, Merge Sort, and Quick Sort work step-by-step. The visual representation makes learning algorithms more intuitive and engaging.
Technologies Used:
HTML, CSS, and JavaScript D3.js (or similar library for animation)
Algorithms:
Bubble Sort, Quick Sort, Merge Sort, Selection Sort, Quick Sort, Insertion Sort
Problem Solved:
This project simplifies the understanding of sorting algorithms by providing a visual, interactive experience. It helps users see the algorithm's progress in real-time, making it easier to grasp the logic behind sorting techniques.
